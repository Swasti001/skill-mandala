# Skill Mandala
A peer-to-peer neural network for skill exchange, habit tracking, and verified learning.

## Features
- **Neural Hub Matching:** Connects teachers and learners based on reciprocal skills.
- **Mandala Tracker:** 31-day visual habit tracker using polar coordinates.
- **Universal Credits:** Blockchain-inspired reputation system.
- **Real-time Sessions:** Chat-based session agreements and scheduling.

## Tech Stack
- Frontend: React, TailwindCSS, Framer Motion
- Backend: Spring Boot, Java, WebSockets
- Database: PostgreSQL
